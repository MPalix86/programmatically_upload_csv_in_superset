General Settings

The dataset name is generated using the CSV file names. If a dataset with the same name already exists in the database, the table will be updated without being deleted and recreated. This allows you to maintain all your charts that use the dataset. If the table does not exist, a new table will be created.



If a dataset with the same name already exists in the database, the table will be deleted and a new table will be created. This does not allow you to maintain all your charts that use the dataset. If the table does not exist, a new table will be created.

Use a JSON model that describes the CSV file you are using. It's assumed the CSV model has the same name as the CSV file you're uploading. So if you're uploading many different CSV files, each CSV must have the corresponding JSON model. If the JSON model is not found, an automatic inference mode is activated to try to infer data types from the CSV.

use the same JSON file for each CSV. (Enable use json model above if you wat to use this option)

Delete files after upload All files will be deleted, including the JSON model files.
